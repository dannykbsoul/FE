### 1.两数之和(1)

```
给定一个整数数组 nums 和一个整数目标值 target，请你在该数组中找出 和为目标值 的那 两个 整数，并返回它们的数组下标。
输入：nums = [2,7,11,15], target = 9
输出：[0,1]
解释：因为 nums[0] + nums[1] == 9 ，返回 [0, 1] 。
```

> 首先考虑先排序然后双指针，但是因为该题目要求返回原数组的下标，所以不能排序
>
> 双重循环也能做到，但是时间复杂度 O(n^2)
>
> 可以考虑用空间复杂度优化时间复杂度，用 O(n)的空间复杂度使得时间复杂度也为 O(n)

```JS
var twoSum = function(nums, target) {
    const map=new Map();
    for(let i=0,len=nums.length;i<len;i++) {
        let findNum=target-nums[i];
        if(map.has(findNum)) return [map.get(findNum),i];
        else map.set(nums[i],i);
    }
};
```

### 2.三数之和(15)

```
给你一个包含 n 个整数的数组 nums，判断 nums 中是否存在三个元素 a，b，c ，使得 a + b + c = 0 ？请你找出所有和为 0 且不重复的三元组。
输入：nums = [-1,0,1,2,-1,-4]
输出：[[-1,-1,2],[-1,0,1]]
```

> 暴力解法，三重循环，时间复杂度 O(n^3)
>
> 因为这题返回的并不是原数组的下标，所以考虑先排序，其中 sort 函数默认是按照字典序进行，即按字母顺序对数组中的元素进行排序，想要对数字进行排列，需要传入比较函数
>
> 接着进行遍历，然后在剩下的序列中进行双指针
> ⚠️ 因为要求三元组不能重复，所以需要加一些条件进行去重

```JS
var threeSum = function(nums) {
    const res=[];
    nums.sort((a,b)=>a-b);
    for(let i=0,len=nums.length;i<len-2;i++) {
        // 去重
        if(i&&nums[i]===nums[i-1]) continue;
        let target = -nums[i],
            l=i+1,
            r=len-1;
        while(l<r) {
            let sum=nums[l]+nums[r];
            if(sum===target) {
                res.push([nums[i],nums[l++],nums[r--]]);
                // 去重
                while(nums[l]===nums[l-1]) l++;
                while(nums[r]===nums[r+1]) r--;
            }
            else if(sum<target) l++;
            else r--;
        }
    }
    return res;
};
```

### 3.最接近的三数之和(16)

```
给定一个包括 n 个整数的数组 nums 和 一个目标值 target。找出 nums 中的三个整数，使得它们的和与 target 最接近。返回这三个数的和。假定每组输入只存在唯一答案。

输入：nums = [-1,2,1,-4], target = 1
输出：2
解释：与 target 最接近的和是 2 (-1 + 2 + 1 = 2) 。
```

> 和上题的三数之和类似

```JS
var threeSumClosest = function(nums, target) {
    let minDis=Infinity,
        res;
    nums.sort((a,b)=>a-b);
    for(let i=0,len=nums.length;i<len-2;i++) {
        if(i&&nums[i]===nums[i-1]) continue;
        let l=i+1,
            r=len-1;
        while(l<r) {
            let sum=nums[i]+nums[l]+nums[r];
            if(sum===target) return target;
            else if(sum<target) {
                if(Math.abs(sum-target)<minDis) {
                    minDis=Math.abs(sum-target);
                    res=sum;
                }
                l++;
            }else {
                if(Math.abs(sum-target)<minDis) {
                    minDis=Math.abs(sum-target);
                    res=sum;
                }
                r--;
            }
        }
    }
    return res;
};
```

### 4.四数之和(18)

```
给定一个包含 n 个整数的数组 nums 和一个目标值 target，判断 nums 中是否存在四个元素 a，b，c 和 d ，使得 a + b + c + d 的值与 target 相等？找出所有满足条件且不重复的四元组。

给定数组 nums = [1, 0, -1, 0, -2, 2]，和 target = 0。

满足要求的四元组集合为：
[
  [-1,  0, 0, 1],
  [-2, -1, 1, 2],
  [-2,  0, 0, 2]
]
```

> if(i&&nums[i]===nums[i-1]) continue; 确保 nums[0]改变了
>
> if(j>i+1&&nums[j]===nums[j-1]) continue; 确保 nums[1]改变了
>
> while(nums[l]===nums[l-1]) l++; 确保 nums[2]改变了
>
> while(nums[r]===nums[r+1]) r--; 确保 nums[3]改变了

```JS
var fourSum = function(nums, target) {
    nums.sort((a,b)=>a-b);
    const res=[];
    for(let i=0,len=nums.length;i<len-3;i++) {
        if(i&&nums[i]===nums[i-1]) continue;
        for(let j=i+1;j<len-2;j++) {
            if(j>i+1&&nums[j]===nums[j-1]) continue;
            let l=j+1,
                r=len-1;
            while(l<r) {
                let sum=nums[i]+nums[j]+nums[l]+nums[r];
                if(sum===target) {
                    res.push([nums[i],nums[j],nums[l++],nums[r--]]);
                    while(nums[l]===nums[l-1]) l++;
                    while(nums[r]===nums[r+1]) r--;
                }else if(sum<target) l++;
                else r--;
            }
        }
    }
    return res;
};
```

### 5.长度最小的连续子数组(209)

```
给定一个含有 n 个正整数的数组和一个正整数 s ，找出该数组中满足其和 ≥ s 的长度最小的 连续 子数组，并返回其长度。如果不存在符合条件的子数组，返回 0。

输入：s = 7, nums = [2,3,1,2,4,3]
输出：2
解释：子数组 [4,3] 是该条件下的长度最小的子数组。
```

> 暴力解法 O(n^2)
>
> 滑动窗口
>
> 我们把数组中的元素不停的入队，直到总和大于等于 s 为止，接着记录下队列中元素的个数，然后再不停的出队，直到队列中元素的和小于 s 为止（如果不小于 s，也要记录下队列中元素的个数，这个个数其实就是不小于 s 的连续子数组长度，我们要记录最小的即可）。接着再把数组中的元素添加到队列中……重复上面的操作，直到数组中的元素全部使用完为止。
>

```JS
var minSubArrayLen = function(s, nums)  {
    let min=Infinity,
        len=nums.length,
        sum=0,
        l=0,
        r=0;
    while(r<len) {
        sum+=nums[r++]
        while(sum>=s) {
            min=Math.min(min,r-l);
            sum-=nums[l++];
        }
    }
    return min===Infinity?0:min;
};
```

> 二分查找
>
> 二分数组的长度，然后以该长度判断是否满足条件，满足的话继续缩小数组长度，否则扩大数组长度
>
> if(i>=size) sum-=nums[i-size];
>
> 用于保证此时sum的和是 <=size个 数的和

~~~JS
var minSubArrayLen = function(s, nums)  {
    let min=0,
        len=nums.length,
        l=1,
        r=len;
    function helper(size) {
        let sum=0;
        for(let i=0;i<len;i++) {
            sum+=nums[i];
            if(i>=size) sum-=nums[i-size];
            if(sum>=s) return true;
        }
    }
    while(l<=r) {
        let mid=Math.floor(l+(r-l)/2);
        if(helper(mid)) {
            min=mid;
            r=mid-1;
        }else l=mid+1;
    }
    return min;
};
~~~

### 6.无重复字符的最长子串(3)

~~~
给定一个字符串，请你找出其中不含有重复字符的 最长子串 的长度。
输入: s = "abcabcbb"
输出: 3 
解释: 因为无重复字符的最长子串是 "abc"，所以其长度为 3。
~~~

> 类似于滑动窗口的改进版
>
> 用map来存放元素出现的下标，当碰到出现过的元素，那么此时窗口的left置为上次出现元素的下标+1，窗口的right每一次迭代都会去计算新的最长子串。

~~~JS
var lengthOfLongestSubstring = function(s) {
    let maxLen=0,
        l=0,
        r=0,
        len=s.length,
        map=new Map();
    while(r<len) {
        let c=s[r];
        if(map.has(c)&&map.get(c)>=l) {
            l=map.get(c)+1;
        }
        maxLen=Math.max(maxLen,r-l+1);
        map.set(c,r++);
    }
    return maxLen;
};
~~~

### 7.最大子序和(53)

~~~
给定一个整数数组 nums ，找到一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。
输入: [-2,1,-3,4,-1,2,1,-5,4]
输出: 6
解释: 连续子数组 [4,-1,2,1] 的和最大，为 6。
~~~

> dp[i]：以nums[i]为结尾的最大子序和
>
> dp[i]=Math.max(dp[i-1]+nums[i],nums[i]);

~~~JS
var maxSubArray = function(nums) {
    let len=nums.length,
        dp=Array(len);
    if(!len) return 0;
    dp[0]=nums[0];
    for(let i=1;i<len;i++) {
        dp[i]=Math.max(dp[i-1]+nums[i],nums[i]);
    }
    return Math.max(...dp);
};
~~~

~~~JS
var maxSubArray = function(nums) {
    let res=nums[0],
        sum=0;
    for(let v of nums) {
        if(sum>0) sum+=v;
        else sum=v;
        res=Math.max(res,sum);
    }
    return res;
};
~~~

### 8.最长公共前缀(14)

~~~
编写一个函数来查找字符串数组中的最长公共前缀。
如果不存在公共前缀，返回空字符串 ""。

输入：strs = ["flower","flow","flight"]
输出："fl"
~~~

> 取数组第一项，从长到短遍历

~~~JS
var longestCommonPrefix = function(strs) {
    if(!strs.length) return "";
    let len=strs.length;
    for(let i=strs[0].length-1;i>=0;i--) {
        let str=strs[0].slice(0,i+1),
            index=1;
        while(index<len) {
            if(strs[index].indexOf(str)!==0) break;
            index++;
        }
        if(index===len) return str;
    }
    return "";
};
~~~

~~~JS
var merge = function(intervals) {
    intervals.sort((a,b)=>a[0]-b[0]);
    for(let i=1,len=intervals.length;i<len;i++) {
        let prev=intervals[i-1],
            cur=intervals[i];
        if(prev[1]>=cur[0]) {
            intervals[i]=[prev[0],Math.max(prev[1],cur[1])];
            intervals[i-1]=false;
        }
    }
    return intervals.filter(x=>x);

var merge = function(intervals) {
    intervals.sort((a,b)=>a[0]-b[0]);
~~~

~~~JS
var addBinary = function(a, b) {
    let res = "",// 保存最后结果
        carry = false;// 保留进位结果
    // 将字符串转换为数组
    num1 = a.split("");
    num2 = b.split("");
    // 当数组的长度都变为0，并且最终不再进位时，结束循环
    while (num1.length || num2.length || carry) {
        // 每次将最后的数字进行相加，使用~~的好处是，即使返回值为 undefined 也能转换为 0
        carry += ~~num1.pop() + ~~num2.pop();
        // 取加法结果的个位加入最终结果
        res = (carry % 2) + res;
        // 判断是否需要进位，true 和 false 的值在加法中会被转换为 1 和 0
        carry = carry > 1;
    }
    // 返回最终结果
    return res;
};
~~~

~~~JS
var multiply = function(num1, num2) {
    let m=num1.length,
        n=num2.length,
        res=Array(m+n).fill(0);
    for(let i=m-1;i>=0;i--) {
        for(let j=n-1;j>=0;j--) {
            let sum=num1[i]*num2[j]+res[i+j+1];

        return res.join("");
    }
};
~~~

### 15.寻找旋转数组中的最小值(153)

~~~
假设按照升序排序的数组在预先未知的某个点上进行了旋转。例如，数组 [0,1,2,4,5,6,7] 可能变为 [4,5,6,7,0,1,2] 。

请找出其中最小的元素。
1 <= nums.length <= 5000
-5000 <= nums[i] <= 5000
nums 中的所有整数都是 唯一 的
nums 原来是一个升序排序的数组，但在预先未知的某个点上进行了旋转。

输入：nums = [3,4,5,1,2]
输出：1
~~~

> 变种二分查找
>
> 因为是旋转数组，所以数组的前半段是有序的，后半段也是有序的，会在分割点处有个断崖
>
> 如果nums[mid]<nums[r]，说明此时mid处于后半有序段中，那么要想找到分割的位置只能在前半部分找，所以二分区间将右边界置为mid
>
> 否则此时nums[mid]>nums[r]，此时mid处于前半有序段中，那么要想找到分割的位置只能在后半部分找，所以二分区间将左边界置为mid+1

~~~JS
var findMin = function(nums) {
    let l=0,
        r=nums.length-1;
    while(l<r) {
        let mid=Math.floor(l+(r-l)/2);
        if(nums[mid]<nums[r]) r=mid;
        else l=mid+1;
    }
    return nums[l];
}
~~~

### 16.比较版本号(165)

~~~
给你两个版本号 version1 和 version2 ，请你比较它们。

版本号由一个或多个修订号组成，各修订号由一个 '.' 连接。每个修订号由 多位数字 组成，可能包含 前导零 。每个版本号至少包含一个字符。修订号从左到右编号，下标从 0 开始，最左边的修订号下标为 0 ，下一个修订号下标为 1 ，以此类推。例如，2.5.33 和 0.1 都是有效的版本号。

返回规则如下：

1 <= version1.length, version2.length <= 500

~~~

~~~JS
var compareVersion = function(version1, version2) {
    const arr1=version1.split(".").map(x=>parseInt(x)),
          arr2=version2.split(".").map(x=>parseInt(x));
    let i=0;
    for(let len1=arr1.length,len2=arr2.length;i<len1||i<len2;i++) {
        if(i>=len1) arr1[i]=0;
        if(i>=len2) arr2[i]=0;
        if(arr1[i]<arr2[i]) return -1;
        else if(arr1[i]>arr2[i]) return 1;
    }
    return 0;
};
~~~

### 17.最大数(179)

~~~
给定一组非负整数 nums，重新排列它们每个数字的顺序（每个数字不可拆分）使之组成一个最大的整数。

注意：输出结果可能非常大，所以你需要返回一个字符串而不是整数。

~~~

> 利用sort比较函数 注意最后要去除前置0

~~~JS
var largestNumber = function(nums) {
    nums.sort((a,b)=>(b+""+a)-(a+""+b));
    let i=0;
    while(i<nums.length) {
        if(nums[i]===0) i++;
        else break;
    }
    nums.splice(0,i);
    if(!nums.length) return "0";
    else return nums.join("");
};
~~~

### 19.阶乘后的零(172)

~~~
给定一个整数 n，返回 n! 结果尾数中零的数量。

输入: 3
输出: 0
解释: 3! = 6, 尾数中没有零。
~~~

> 乘积产生0主要是依靠2*5得到，所以我们要尽量的去找因数2和5，但是发现5比2大，所以2的次数肯定要比5多，所以我们只需要找5为因数的个数
>
> 所以我们可以遍历每一个数，然后对5取余

~~~JS
var trailingZeroes = function(n) {
    let count=0;
    for(let i=1;i<=n;i++) {
        let N=i;
        while(N>=5) {
            if(N%5===0) {
                N/=5;
                count++;
            }else break;
        }
    }
    return count;
};


// 不断除以 5, 是因为每间隔 5 个数有一个数可以被 5 整除, 然后在这些可被 5 整除的数中, 每间隔 5 个数又有一个可以被 25 整除, 故要再除一次, ... 直到结果为 0, 表示没有能继续被 5 整除的数了.
var trailingZeroes = function(n) {
    let count=0;
    while(n>=5) {
        n=parseInt(n/5);
        count+=n;
    }
    return count;
};
~~~

### 20.数组中的第K个最大元素(215)

~~~
在未排序的数组中找到第 k 个最大的元素。请注意，你需要找的是数组排序后的第 k 个最大的元素，而不是第 k 个不同的元素。

输入: [3,2,1,5,6,4] 和 k = 2
输出: 5
~~~

> 方法一：快速排序+二分法
>
> partition（切分）操作总能排定一个元素，还能够知道这个元素它最终所在的位置，这样每经过一次 partition（切分）操作就能缩小搜索的范围，这样的思想叫做 “减而治之”
>
> 注意为啥要用shuffle，为了使得每次快速排序选取的基准点都是比较随机，防止快速排序的性能从O(nlogn)退化到O(n^2)

~~~JS
var findKthLargest = function(nums, k) {
    let len=nums.length,
        l=0,
        r=len-1;
    shuffle(nums);
    while(l<=r) {
        let index=partiton(nums,l,r);
        if(index<k-1) l=index+1;
        else if(index>k-1) r=index-1;
        else return nums[index];
    }
};

function shuffle(nums) {
    for(let i=nums.length;i;i--) {
        let random=Math.floor(Math.random()*i);
        [nums[i-1],nums[random]]=[nums[random],nums[i-1]];
    }
}

function partiton(nums,l,r) {
    let privot=nums[l],
        index=l;
    for(let i=l+1;i<=r;i++) {
        if(nums[i]>privot) {
            index++;
            [nums[index],nums[i]]=[nums[i],nums[index]];
        }
    }
    [nums[l],nums[index]]=[nums[index],nums[l]];
    return index;
}
~~~

~~~JS
var isAnagram = function(s, t) {
    if(s.length!=t.length) return false;
    const map=new Map();
    for(let i=0,len=s.length;i<len;i++) {
        map.set(s[i],(map.get(s[i])||0)+1);
        map.set(t[i],(map.get(t[i])||0)-1);
    }
    for(let k of map.values()) {
        if(k) return false;
    }
    return true;
};
~~~
